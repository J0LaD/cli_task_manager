# Simple Rust CLI Task Manager

A lightweight, interactive command-line application written in Rust to manage a simple "To-Do" list. This project is a great demonstration of basic Rust concepts like standard I/O operations, vector manipulation, loops, and pattern matching.

## Features

- **Interactive Menu**: Stays open to receive multiple commands without restarting.
- **Add Tasks**: Continuously add new tasks to your list.
- **Delete Tasks**: Safely remove specific items by typing their exact name.
- **Display List**: View all your current tasks formatted cleanly.
- **In-Memory Storage**: Tasks are stored temporarily during the session (cleared upon exit).

## Prerequisites

Make sure you have [Rust and Cargo](https://rustup.rs/) installed on your machine.
